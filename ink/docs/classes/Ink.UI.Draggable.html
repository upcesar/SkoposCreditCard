Constructor

Ink.UI.Draggable

(
  • target
  • [options]
)

Defined in Ink/UI/Draggable/1/lib.js:19

Parameters:

  • target String | DOMElement

    Target element.

  • [options] Object optional

    Optional object for configuring the component

    • [constraint] String optional

      Movement constraint. None by default. Can be vertical, horizontal, or both.

    • [constraintElm] String | DomElement optional

      Constrain dragging to be within this element. None by default.

    • [top,left,right,bottom] Number optional

      Limits for constraining draggable movement.

    • [handle] String | DOMElement optional

      if specified, this element will be used as a handle for dragging.

    • [revert] Boolean optional

      if true, reverts the draggable to the original position when dragging stops

    • [cursor] String optional

      cursor type (CSS cursor value) used when the mouse is over the draggable object

    • [zIndex] Number optional

      zindex applied to the draggable element while dragged

    • [fps] Number optional

      if defined, on drag will run every n frames per second only

    • [droppableProxy] DomElement optional

      if set, a shallow copy of the droppableProxy will be put on document.body with transparent bg

    • [mouseAnchor] String optional

      defaults to mouse cursor. can be 'left|center|right top|center|bottom'

    • [dragClass='drag'] String optional

      class to add when the draggable is being dragged.

    • [onStart] Function optional

      callback called when dragging starts

    • [onEnd] Function optional

      callback called when dragging stops

    • [onDrag] Function optional

      callback called while dragging, prior to position updates

    • [onChange] Function optional

      callback called while dragging, after position updates

Example:

Ink.requireModules( ['Ink.UI.Draggable_1'], function( Draggable ){
    new Draggable( '#myElementId' );
});
