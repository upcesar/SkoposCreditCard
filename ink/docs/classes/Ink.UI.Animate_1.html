static

Defined in Ink/UI/Animate/1/lib.js:102

Animate a div using one of the animate.css classes

Note: This is a utility method inside the Animate class, which you can access through Animate.animate(). Do not mix these up.

Parameters:

  • element DOMElement

    animated element

  • animation String

    animation name

  • [options] Object optional
    • [onEnd=null] Function optional

      callback for animation end

    • [removeClass=false] Boolean optional

      whether to remove the Css class when finished

    • [duration=medium] String | Number optional

      duration name (fast|medium|slow) or duration in ms

Example:

 Animate.animate(myDiv, 'shake', {
     onEnd: function () {
         alert('Finished shaking!');
     }
 });
