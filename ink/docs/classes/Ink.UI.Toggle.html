Toggle component

Constructor

Ink.UI.Toggle

(
  • selector
  • [options]
)

Defined in Ink/UI/Toggle/1/lib.js:9

Parameters:

  • selector String | DOMElement
  • [options] Object optional

    Options

    • target String

      CSS Selector that specifies the elements that this component will toggle

    • [classNameOn='show-all'] String optional

      className when toggle is On

    • [classNameOff='hide-all'] String optional

      className when toggle is Off.

    • [triggerEvent='click'] String optional

      Event that will trigger the toggling.

    • [closeOnClick=true] Boolean optional

      When this is on, if the user clicks outside of the toggled content, the target is toggled off.

    • [closeOnInsideClick='a[href] Selector optional

      Toggle off when an element matching this selector is clicked. Set to null to deactivate the check. Default: 'a[href]' (finds links)

    • [initialState=null] Boolean optional

      Whether to start toggled off, on, or as found in the markup. (false: off, true: on, null: markup)

    • [onChangeState=null] Function optional

      Callback to be called when the toggle state changes. Return false to cancel the event.

Example:

 <div class="ink-dropdown">
     <button class="ink-button toggle" data-target="#dropdown">Dropdown <span class="icon-caret-down"></span></button>
     <ul id="dropdown" class="dropdown-menu">
         <li class="heading">Heading</li>
         <li class="separator-above"><a href="#">Option</a></li>
         <li><a href="#">Option</a></li>
         <li class="separator-above disabled"><a href="#">Disabled option</a></li>
         <li class="submenu">
             <a href="#" class="toggle" data-target="#submenu1">A longer option name</a>
             <ul id="submenu1" class="dropdown-menu">
                 <li class="submenu">
                     <a href="#" class="toggle" data-target="#ultrasubmenu">Sub option</a>
                     <ul id="ultrasubmenu" class="dropdown-menu">
                         <li><a href="#">Sub option</a></li>
                         <li><a href="#" data-target="ultrasubmenu">Sub option</a></li>
                         <li><a href="#">Sub option</a></li>
                     </ul>
                 </li>
                 <li><a href="#">Sub option</a></li>
                 <li><a href="#">Sub option</a></li>
             </ul>
         </li>
         <li><a href="#">Option</a></li>
     </ul>
 </div>
