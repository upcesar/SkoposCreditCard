create

(
  • tag
  • properties
)

Defined in Ink/Dom/Element/1/lib.js:57

Creates a DOM element

Parameters:

  • tag String

    tag name

  • properties Object

    object with properties to be set on the element. You can also call other functions in Ink.Dom.Element like this

Example:

 var myPanel = InkElement.create('div', {
     appendHTML: '<span></span>',
     className: 'classes to add',
     insertAfter: anotherPane
 };

 myPanel.innerHTML;  // -> '<div class="classes to add"><span></span></div>
 myPanel.parentNode;  // -> anotherPane

unwrap

(
  • elem
  • [wrapperSelector]
)

Defined in Ink/Dom/Element/1/lib.js:1439

Pulls an element out of its wrapper.

Parameters:

  • elem Object

    The element you're trying to unwrap. This should be a child of the wrapper.

  • [wrapperSelector] String optional

    Use this if you want to find your wrapper (if elem is inside several layers of wrappers).

Example:

When you have this:

 <div id="wrapper">
     <div id="unwrapMe"></div>
 </div>

If you do this:

 InkElement.unwrap('unwrapMe');

You get this:

 <div id="unwrapMe"></div>
 <div id="wrapper"></div>

wrap

(
  • target
  • container
)

Defined in Ink/Dom/Element/1/lib.js:1398

Wraps an element inside a container.

The container may or may not be in the document yet.

Parameters:

  • target String | DomElement

    Element to be wrapped

  • container String | DomElement

    Element to wrap the target

Returns:

Container element

Example:

before:

<div id="target"></div>

call this function to wrap #target with a wrapper div.

InkElement.wrap('target', InkElement.create('div', {id: 'container'});

after: 

<div id="container"><div id="target"></div></div>
