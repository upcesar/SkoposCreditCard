Parameters:

  • selector String | DOMElement
  • options Object

    Options

    • [size] Number optional

      number of pages.

    • [totalItemCount] Number optional

      Total numeber of items to display

    • [itemsPerPage] Number optional

      number of items per page.

    • [maxSize] Number optional

      If passed, only shows at most maxSize items. displays also first|prev page and next page|last buttons

    • [start] Number optional

      start page. defaults to 1

    • [firstLabel] String optional

      label to display on first page button

    • [lastLabel] String optional

      label to display on last page button

    • [previousLabel] String optional

      label to display on previous button

    • [nextLabel] String optional

      label to display on next button

    • [previousPageLabel] String optional

      label to display on previous page button

    • [nextPageLabel] String optional

      label to display on next page button

    • [onChange] Function optional

      optional callback. Called with (thisPaginator, newPageNumber).

    • [hashParameter] String optional

      parameter to use on setHash. by default uses 'page'

    • [parentTag] String optional

      HTML Tag used as the parent node.

    • [childTag] String optional

      HTML Tag used as the child nodes.

    • [wrapperClass] String optional

      CSS Class used in the wrapper element

    • [paginationClass] String optional

      CSS Class used in the pagination element

    • [activeClass] String optional

      CSS Class used to mark page as active

    • [disabledClass] String optional

      CSS Class used to mark page as disabled

    • [hideClass] String optional

      CSS Class used to hide elements

    • [previousClass] String optional

      CSS Class used in the previous element

    • [previousPageClass] String optional

      CSS Class used in the previous page element

    • [nextClass] String optional

      CSS Class used in the next element

    • [nextPageClass] String optional

      CSS Class used in the next page element

    • [numberFormatter] Function optional

      optional function which takes and 0-indexed number and returns the string which appears on a numbered button
