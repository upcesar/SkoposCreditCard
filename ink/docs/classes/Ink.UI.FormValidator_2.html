)

Defined in Ink/UI/FormValidator/2/lib.js:623

Parameters:

  • selector String | DOMElement

    Either a CSS Selector string, or the form's DOMElement

  • [options.eventTrigger='submit'] String optional

    What event do we listen for.

  • [options.neverSubmit=false] Boolean optional

    Always cancel the event? Use this to avoid submitting the form.

  • [options.searchFor='input, Selector

    select, textarea, .control-group'] Look in these inputs for validation data-attributes.

  • [options.beforeValidation] Function optional

    Callback to be executed before validating the form

  • [options.onError] Function optional

    Validation error callback

  • [options.onSuccess] Function optional

    Validation success callback

Example:

Ink.requireModules( ['Ink.UI.FormValidator_2'], function( FormValidator ){
    var myValidator = new FormValidator( '#my-form' );
});
