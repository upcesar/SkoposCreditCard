Ink.UI.SortableList Class

Defined in: Ink/UI/SortableList/1/lib.js:12
Module: Ink.UI.SortableList_1

Adds sortable behaviour to any list!

Constructor

  • [options.draggedClass='hide-all']
  • [options.draggingClass='dragging']
  • [options.dragSelector='li']
  • [options.handleSelector=null]
  • [options.moveSelector=null]

  • [options.cancelMouseOut=false]
)

Defined in Ink/UI/SortableList/1/lib.js:12

Parameters:

  • selector String | DOMElement
  • [options.placeholderClass='placeholder'] String optional

    CSS class to be added to the "ghost" element being dragged around

  • [options.draggedClass='hide-all'] String optional

    CSS class added to the original element.

  • [options.draggingClass='dragging'] String optional

    CSS class added to the html element when the user is dragging.

  • [options.dragSelector='li'] String optional

    CSS selector for the drag enabled nodes.

  • [options.handleSelector=null] String optional

    CSS selector for the drag handle. If present, you can only drag nodes by this selector.

  • [options.moveSelector=null] String optional

    CSS selector to validate a node move. If present, you can only move nodes into this selector.

  • [options.swap=false] Boolean optional

    Flag to swap moving element with target element instead of changing its order.

  • [options.cancelMouseOut=false] Boolean optional

    Flag to cancel moving if mouse leaves the container element.

Example:

 <ul class="unstyled ink-sortable-list" id="slist" data-handle-selector=".ink-label">
     <li><span class="ink-label info">drag here</span>primeiro</li>
     <li><span class="ink-label info">drag here</span>segundo</li>
     <li><span class="ink-label info">drag here</span>terceiro</li>
 </ul>
 <script>
     Ink.requireModules( ['Ink.Dom.Selector_1','Ink.UI.SortableList_1'], function( Selector, SortableList ){
         var sortableListElement = Ink.s('.ink-sortable-list');
         var sortableListObj = new SortableList( sortableListElement );
     });
 <\/script>
