get

  • name
)
String | Object public static

Defined in Ink/Util/Cookie/1/lib.js:19

Gets an object with current page cookies

Parameters:

  • name String

Returns:

String | Object:

If the name is specified, it returns the value related to that property. Otherwise it returns the full cookie object

Example:

Ink.requireModules(['Ink.Util.Cookie_1'], function( InkCookie ){
    var myCookieValue = InkCookie.get('someVarThere');
    console.log( myCookieValue ); // This will output the value of the cookie 'someVarThere', from the cookie object.
});

Sets a cookie

Parameters:

  • name String

    Cookie name

  • value String

    Cookie value

  • [expires] Number optional

    Number to add to current Date in seconds

  • [path] String optional

    Path to sets cookie (default '/')

  • [domain] String optional

    Domain to sets cookie (default current hostname)

  • [secure] Boolean optional

    True if wants secure, default 'false'

Example:

Ink.requireModules(['Ink.Util.Cookie_1'], function( InkCookie ){
    var expireDate = new Date( 2014,00,01, 0,0,0);
    InkCookie.set( 'someVarThere', 'anyValueHere', expireDate.getTime() );
});
